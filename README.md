# Introduction to Python

A comprehensive Jupyter notebook covering Python from basics to GPU computing — designed for use with Red Hat OpenShift AI.

## Topics Covered

1. **Python Basics** — `print`, variables, data types
2. **Control Flow** — loops, conditionals
3. **Functions** — definitions, return values
4. **Data Structures** — lists, dictionaries
5. **OOP** — classes, methods
6. **Advanced Functions** — `*args`, `**kwargs`, decorators
7. **Modules & Imports**
8. **File Operations** — reading, writing, error handling
9. **Cloud Storage (S3)**
10. **NumPy**
11. **Data Visualization** — line plots, bar charts, heatmaps, 3D
12. **GPU Computing** — detection, benchmarking

## Getting Started

Open `Python_Introduction.ipynb` in JupyterLab or any notebook environment and run the cells sequentially.
